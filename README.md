# Home-Assistant
Tento kód slouží k měření teploty pomocí digitálního teplotního senzoru Dallas DS18B20. ESP32 čte teplotu a následně ji přes Wi-Fi odesílá pomocí MQTT protokolu do Home Asistantu. 
# Soupis použitých komponent
ESP32 super mini c3  
Teplotní čidlo <a href="https://dratek.cz/arduino-platforma/1187-teplotni-senzor-digitalni-dallas-ds18b20.html">Dallas DS18B20</a></li>
Rezistor 4,7kΩ  
Senzor hořlavých plynů MQ2 (Čip: LM393,Sensor: ZYMQ)
